# yg-double-search

Google Extension for searching with Yahoo! JAPAN, and Google simultaneously.

## Preparations

- Excute command

```bash
yarn install
yarn webpack
```

- Open Chrome
- Manage Extensions
- Turn on developer mode
- Load unpacked
- Open 'yg-double-search/dist'

Ex.) You can search 台風 with Yahoo! JAPAN and Google to compare UX.
